CEEN 514 Geoids


We are The Geoids! Pictured from left to right: Tyler Mickelson, Luke Newey, Joseph Browning. This travel time polygons app was developed as a term project for our masters class at Brigham Young University. Under the direction of the prolific and most honorable Dr. Dan Pepper Ames, we learned how to code and produce this finished product over the course of the semester.

  • Tyler worked on developing the geoprocessing model that runs the main functon of our app. This included finding a Utah roadway network GIS database and then using it to create a model that calculates travel time polygons.
  • Luke worked on the development of the rescaffolded app and the incorporation of the code from the previous app along with the general structure and appearance of the app. He also was the keeper of the Github keys and was in charge of setting up and maintaining the repositories.
  • Joseph worked on the Javascript code to connect the map functions to the user interface. For example, when you click "Calculate" the app would not have done anything without Joseph. He is also the Data Master of this project; handling the necessary data to make this app possible.


We want to acknowledge the Utah AGRC Data Portal for providing the road network that is used in calculating the travel time polygon. In this model each roadway and intersection was assigned an impedance value in minutes per unit length. These values were assumed based off of the free flow speed of each roadway under normal traffic conditions. Therefore, travel time polygons under congested conditions would likely be smaller than the results found in this app. More details on the code and scaffolding of this app can be found on the Geoids github repository .
